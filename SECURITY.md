# Security policy

This repository is a synthetic showcase and must not contain real applicant data or credentials.

Do not report real job-board credentials, API keys, resumes, application screenshots, browser profiles, or candidate records in a public issue. If you discover a security or privacy issue, contact the repository owner privately through the contact method listed on their GitHub profile.

Before opening a pull request, run:

```bash
npm run privacy:check
npm test
```
